# Bootstrap

Bootstrap encompasses minimal setup guides for quickly establishing the agent development workspace with core repositories and essential tooling.

This domain covers:

- **Workspace Creation**: Creating workspace directory and cloning core repositories
- **IDE Integration**: Setting up Amazon Q plugin and workspace rules
- **Minimal Configuration**: Essential setup to get started quickly
- **Quick Start**: Getting from zero to working development environment

Bootstrap guides provide the fastest path to a working agent development workspace with minimal steps and configuration.
